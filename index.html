iMuze API, version codename Joplin

How To Use the iMuze API

Joplin is the last version of the iMuze API. It tlets you generate music on demand. To generate music you just need to specify the genre, sub-genre, duration and the structure of the music you want.

API base url

The base url for this version of the API is 
https://joplin.imuze.io

Authenticate

First request an account on iMuze.io . Use the following route to authenticate:

POST tokens

The json payload should include your account credentials

{
    "email": "you@domain.tld  ",
    "password": "my_password"
}

With the correct credentials, the response is a valid API token:

{
    "token": "gergdhvdb4399ewnfusrodfd....t45"
}

Generate Music

Use the following route to generate music:

POST /musics

The json payload should include the genre, subgenre, duration (in ms), and the structure of the music specified as a set of movements.

{
    "genre": "hiphop",
    "subgenre": "dirt",
    "duration_ms": 31000,
    "movements": [
        {
            "start": 0,
            "duration": 0.33,
            "energy": "calm"
        },
        {
            "start": 0.33,
            "duration": 0.33,
            "energy": "medium"
        },
        {
            "start": 0.66,
            "duration": 0.33,
            "energy": "dynamic"
        }
    ]
}

the "movement" property contains:

  • start : time percentage of the music where the movement start
  • duration : movement duration in percentage of the whole
  • energy : defines the relative energy level of the movements, can be one of the following ['calm', 'medium', 'dynamic']

NB: you must provide the token in the Authorization Header

The response describes the music, it's structure in milliseconds and the URI to download the mp3:

{
  "id": "3f782e98-613d-4f68-b27f-05237b90ba65",
  "duration_ms": 32000,
  "measure": "4/4",
  "genre": "hiphop",
  "music_engine_version": "b2c2ce7",
  "movements": [
    {
      "energy": "calm",
      "start_ms": 0,
      "duration_ms": 10666.666666666666
    },
    {
      "energy": "medium",
      "start_ms": 10666.666666666666,
      "duration_ms": 10666.666666666666
    },
    {
      "energy": "dynamic",
      "start_ms": 21333.333333333332,
      "duration_ms": 10666.666666666666
    }
  ],
  "mp3_uri": "//joplin.imuze.io/musics/3f782e98-613d-4f68-b27f-05237b90ba65/mp3",
  "bpm": 90,
  "subgenre": "dirt"
}

Enjoy your music

Use the mp3_uri to download and listen to your music, enjoy !
